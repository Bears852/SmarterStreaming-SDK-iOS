大牛直播iOS端SDK

## CocoaPods

```ruby
# 拉流播放器
pod 'SmarterStreaming/Player'
# 推流
pod 'SmarterStreaming/Publisher'
# 推流 - 无美颜版 (未集成GPUImage)
pod 'SmarterStreaming/Publisher_NO_GPUImage'
```
